# ⚡ QUICK START - Chỉ 3 Bước!

## 🎯 Bước 1: Setup Gmail OAuth (Lần Đầu)

### A. Tạo Google Cloud Project
- Vào: https://console.cloud.google.com
- Tạo project: "TeacherBot"
- Bật: Gmail API
- Tạo: OAuth 2.0 Web Credentials
- Thêm Redirect URI:
  ```
  http://localhost:5000/api/email/oauth2callback
  ```

### B. Copy Client ID & Secret
- Copy `Client ID` từ Google Cloud Console
- Copy `Client Secret` từ Google Cloud Console

### C. Tạo `.env` file
```powershell
# CMD hoặc PowerShell
echo GMAIL_CLIENT_ID=PASTE_YOUR_ID > .env
echo GMAIL_CLIENT_SECRET=PASTE_YOUR_SECRET >> .env
```

## ❓ Gỡ Lỗi Nhanh

### Lỗi: "Gmail chưa được cấu hình"
- Kiểm tra `.env` file có `GMAIL_CLIENT_ID` không
- Restart server nếu đã thêm

### Lỗi: "Chưa đăng nhập Gmail" khi vào tab Email
- Click "Đăng nhập Gmail" button
- Chọn tài khoản, cấp quyền

### Lỗi: Port 5000 đang sử dụng
```powershell
# Tìm process sử dụng port 5000
Get-NetTCPConnection -LocalPort 5000

# Hoặc chạy server trên port khác
python app.py  # sửa port trong .env hoặc code
```

### Lỗi: ModuleNotFoundError
```powershell
# Cài lại dependencies
pip install -r requirements.txt
```

### Lỗi: "CORS error" hoặc "Connection refused"
- Kiểm tra server chạy không: `curl http://localhost:5000/api/health`
- Nếu không chạy: Chạy `.\run_app.ps1` lại

## 💡 Tips & Tricks

1. **Muốn thay đổi port?**
   ```powershell
   $env:FLASK_RUN_PORT=8000
   python app.py
   ```

2. **Muốn reset database?**
   ```powershell
   Remove-Item data -Recurse -Force
   python setup_env.py
   ```

3. **Muốn xem logs chi tiết?**
   - Terminal sẽ hiển thị logs
   - Hoặc check DevTools (F12) trong browser

4. **Muốn đổi Gmail account?**
   - Click "Đăng xuất Gmail"
   - Click "Đăng nhập Gmail" 
   - Chọn account khác

5. **Muốn thêm AI provider?**
   - Lấy API key từ provider
   - Thêm vào `.env`
   - Restart server
